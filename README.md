# Java Server Pages

# Sobre o projeto

O projeto full-stack foi desenvolvido utilizando Servlets com JSP, JDBC com SQL, Java, JavaScript, Jquery, Bootstrap, Jasper para relatórios e Apache Poi para gerar planilha excel.
